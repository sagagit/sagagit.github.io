LYSON - обладнання для бджільництва


Про нас

Фірма LYSON – це європейський лідер виробництва обладнання для бджільництва. Обладнання, яке пропонує LYSON є передовим у своїй галузі. Серед асортименту обладнання найбільшим попитом користуються медогонки, обладнання для кремування меду, столи для розпечатування рамок, обладнання для фасування меду, насоси для перекачування меду, преси для забрусу та сушки для пилку. Обладнання виготовляється як для невеликих пасік так і для промислових. Фірма LYSON пропонує широкий асортимент продукції. Крім обладнання ви можете обрати вулики, захисний одяг, роздільні решітки, димарі, товари для виведення маток та багато іншого. Тобто все, що потрібно для роботи на пасіці можна знайти в асортименті LYSON.
